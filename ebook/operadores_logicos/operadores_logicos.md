# Operadores Lógicos

Os operadores lógicos (também conhecidos como operadores booleanos) são utilizados para auxiliar a criar estruturas de decisão.

Em Python eles são escritos por extenso (em inglês):

|Operador booleano |Python
|--|--
|e |and
|ou |or
|negação |not
